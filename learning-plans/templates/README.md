## How to Use Templates

### Option 1: Via `/learning-plan` Skill
Run `/learning-plan` and select a template when prompted. The skill will:
1. Ask about your context (company, time available)
2. Customize the template to your situation
3. Generate personalized module files

### Option 2: Self-Directed Study
1. Choose a template that matches your goals
2. Work through modules in order (1-2 per week recommended)
3. Complete exercises and reflection questions
4. Use `/learn [topic]` for deeper dives on specific concepts

## Template Structure

Each template includes:
- **Target audience** - Who this path is designed for
- **Learning outcomes** - What you'll be able to do after completion
- **Time commitment** - Total hours and recommended pace
- **Modules** with:
  - Learning objectives
  - Core episodes (with guest, title, and focus)
  - Key frameworks (linked to frameworks library)
  - Suggested exercises
- **Customization notes** - How to adapt based on your context

## Creating Custom Plans

If none of these templates fit your needs, run `/learning-plan` and select "custom" to create a fully personalized curriculum based on your specific goals, time, and context.
